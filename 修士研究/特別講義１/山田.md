# HTMLの歴史
- WHATWG HTMLを作ってる奴ら
  - ブラウザでアプリを作りたいと　提案したもの　→ HTML5
- HTML Living Standardに一本化

## HTML５の変更点
- 文章構造の明示（nav, article)
- マルチメディア対応(audio,video)
- グラフィック機能（canvas,SVGの埋め込み）
- フォームの機能強化（inout,type）
- Offline Application Caching
- クライアントサイドストレージ（localstorage）
- 編集機能（contenteditable属性）
- 多くのAPI
